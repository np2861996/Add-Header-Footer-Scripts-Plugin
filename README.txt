== Description ==

Just install Plugin and enjoy feature of Add Header Footer Scripts Plugin.

This plugin used for add header and footer scripts.

1). Go to add header and footer scripts in the left side options list in plugin.

2). There will be three options available,Scripts in Header, Scripts in Body and Scripts in Footer.

3). Add your scripts, Save And it's Done. Simple Right!!

== Installation ==
1. Upload the entire `insertadd-header-footer-scripts` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).
